Probability & Random Variables – Theory

1. Random Experiment

A random experiment is an experiment in which the outcome cannot be predicted with certainty in advance.

Examples include tossing a coin, rolling a die, or drawing a card from a deck.

2. Sample Space

The sample space, denoted by S, is the set of all possible outcomes of a random experiment.

Each element of the sample space represents a possible result.

3. Event

An event is any subset of the sample space.

An event may consist of one outcome or multiple outcomes.

4. Probability of an Event

The probability of an event A is defined as the ratio of favourable outcomes to total possible outcomes.

P(A) = Number of favourable outcomes / Total outcomes

5. Complement of an Event

The complement of an event A consists of all outcomes in the sample space that are not in A.

P(Ac) = 1 − P(A)

6. Mutually Exclusive Events

Two events are said to be mutually exclusive if they cannot occur at the same time.

Their intersection is empty.

P(A ∩ B) = 0

7. Independent Events

Two events are independent if the occurrence of one does not affect the occurrence of the other.

P(A | B) = P(A)

8. Addition Law of Probability

For overlapping events, the probability of occurrence of either event is:

P(A ∪ B) = P(A) + P(B) − P(A ∩ B)

For mutually exclusive events:

P(A ∪ B) = P(A) + P(B)

9. Multiplication Law of Probability

For dependent events:

P(A ∩ B) = P(A) P(B | A)

For independent events:

P(A ∩ B) = P(A) P(B)

10. Conditional Probability

Conditional probability is the probability of an event occurring given that another event has already occurred.

P(A | B) = P(A ∩ B) / P(B)

11. Random Variables

A random variable assigns a numerical value to each outcome of a random experiment.

It allows probabilistic outcomes to be analyzed mathematically.

12. Discrete Random Variables

Discrete random variables take countable values.

Examples include number of defective items or number of heads in coin tosses.

13. Probability Mass Function (PMF)

The PMF gives the probability that a discrete random variable takes a specific value.

P(X = xi)

14. Cumulative Distribution Function (CDF)

The CDF gives the probability that a random variable is less than or equal to a given value.

F(x) = P(X ≤ x)

15. Binomial Distribution

The binomial distribution represents the number of successes in n independent trials.

P(X = k) = (n choose k) pk (1 − p)n−k

E[X] = np, Var(X) = np(1 − p)

16. Hypergeometric Distribution

This distribution applies when sampling is done without replacement.

P(X = k) = [(K choose k)(N−K choose n−k)] / (N choose n)

17. Poisson Distribution

The Poisson distribution models rare events occurring in a fixed interval.

P(X = k) = (λk e−λ) / k!

E[X] = Var(X) = λ

18. Continuous Random Variables

Continuous random variables can take any value in a given interval.

They are described using probability density functions.

19. Probability Density Function (PDF)

The PDF represents the density of probability over a range of values.

∫ f(x) dx = 1

20. Normal Distribution

The normal distribution is a continuous distribution characterized by mean and variance.

f(x) = (1 / √(2πσ²)) e−(x − μ)² / (2σ²)

21. Standard Normal Variable

Standardization converts a normal variable into a standard normal variable.

Z = (X − μ) / σ

22. Expectation

The expected value represents the long-run average of a random variable.

E[X] = Σ xiP(X = xi)

E[X] = ∫ x f(x) dx

23. Variance

Variance measures the dispersion of a random variable around its mean.

Var(X) = E[X²] − (E[X])²

24. Total Probability Theorem

This theorem is used when an event can occur due to several mutually exclusive causes.

P(B) = Σ P(B | Ai)P(Ai)

25. Bayes’ Theorem

Bayes’ theorem is used to find reverse conditional probabilities.

P(Ai | B) = [P(B | Ai)P(Ai)] / Σ P(B | Aj)P(Aj)

26. Probability of Error

Probability of error occurs when the received signal differs from the transmitted signal.

It is widely used in communication systems and artificial intelligence.
